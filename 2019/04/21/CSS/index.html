CSS简介

什么是CSS

CSS，层叠样式表，是一组样式设置规则，用于空中页面的外观样式

为什么使用CSS

  • 实现内容和样式分离，便于团队开发
  • 样式复用，便于网站的后期维护
  • 可以实现页面的精确控制，可以让页面变得更精美

CSS作用

  • 页面外观美化
  • 布局和定位

CSS基本用法

CSS语法

  • 选择器：要修饰的对象（东西）
  • 属性：修饰对象的哪个属性（样式）
  • 属性值：样式的取值

CSS应用方式

也称为CSS引用方式，有三种方式：内部样式，行内样式，外部样式

内部样式

也称为内嵌样式，在页面头部通过style标签来定义

对当前页面所有符合样式选择器的标签能起作用

行内样式

也称为嵌入样式，使用HTML标签的style属性定义

只对设置style属性的标签起作用

外部样式

使用单独.css文件来定义样式，然后在页面中使用link标签，或者@import指令引入

  • 使用link标签链接外部样式文件（推荐）

    type属性可以省略

    link本质上是链接

  • @import指令导入外部样式表文件

    impot本质上是导入，效率低

选择器

基础选择器

标签选择器

也称为元素选择器，使用HTML标签作为选择器名称

也标签名作为样式应用依据

类选择器

使用自定义名称，以.号为前缀，然后通过HTML标签属性调用类选择器

以标签的class属性作为样式应用的依据

注意事项：

  • 调用时不能添加.号
  • 同时调用多个类选择器中间以空格隔开
  • 类选择名不能以数字开头

ID选择器

使用自定义名称，以#号作为前缀，然后通过HTML标签的id助兴进行名称匹配

以标签的id属性作为样式应用依据，一对一关系

复杂选择器

复合选择器

比如：标签选择器和类选择器，标签选择器和ID选择器，一起使用

必须同时满足两个条件才能应用样式

组合选择器

也称为集体声明

将多个具有相同样式的选择器进行声明，使用,隔开

嵌套选择器

在某个选择器内在设置选择器，通过空格隔开

只用满足层次关系最里层的选择器所对应的的标签才会应用该样式

注：使用空格是不区分父子还是后代，使用CSS中新增的>时必须是父子关系才可以

伪类选择器

根据不同状态来显示不同样式，一般多用于a标签

四种状态：

  • link：未被访问过的链接
  • visited：已经被访问过的链接
  • hover：鼠标移动到的连接上，即悬浮在哪
  • :active：当前正在选定的链接，即激活的链接

注：默认超链接为蓝色有下划线

伪元素选择器

  • first-letter：为第一个字符添加样式

  • firs-line：为第一行添加样式

  • :before：在元素内容最前面去添加其他内容，需要配合content属性使用

  • :after：在元素内容最后面去添加其他内容，需要配合content属性使用

选择器优先级

优先级

ID选择器>类选择器>标签选择器

原因：首先记载标签选择器，再加载类选择器，然后加载ID选择器，最后加载行内样式

内外部样式加载顺序

就近原则

原因：按照书写顺序一次生效，在同优先级的前提下，后加载的会覆盖先加载的同名样式，所以离得越近越优先

!important

可以使用!important使某个样式具有最高的优先级

盒子模型

简介

盒子模型是网页布局的基础，将页面所有的元素都看作成一个盒子，盒子包含以下几个属性：

  • height：高度
  • width：宽度
  • border：边框
  • padding：内边距，内容距离边框之间的距离
  • margin：外边距，盒子和盒子之间的距离

盒子属性

border

表示盒子的边框

  • 分为四个方向：

​ top（上），right（右），bottom（下），left（左）

​ border-top，border-right，border-bottom，border-left

  • 每个边都包含三种样式：

​ color（颜色），width（粗细），style（样式）

样式style的取值：

​ solid（实线），dashed（虚线），dotted（点线），double（双线），inset（内嵌3D线），outset（外嵌3D线）

简写：三种方式

  • 按方向简写：

    border-top，border-right，border-bottom，border-left

    书写顺序：border-方向：width style color

  • 按样式简写：

    border-color，border-width，border-style

    书写顺序：border-样式：top right bottom left

    必须按顺时针方向书写，同时，是可以缩写的

    ​ border-width:2px; —-> 四个边的宽度都是2px

    ​ border-width:2px,4px —> 上下一样为2px，左右一样为4px

    ​ border-width:2px,3px,4px —> 上面为2px，右面为3px，下面为4px，左面为3px

    规则：如果省略，则认为上下一样，左右一样

  • 终极简写

    如果四边完全相同，border：width style color

padding

表示盒子内边距，及内容与边框之间的距离

同样也分为四个方向，也可以简写（按顺时针方向，上下一样，左右一样）

注：如果上下冲突，以上为准，如果左右冲突，以左为准

margin

表示盒子的外边距，即盒子与盒子之间的距离

同样也分为四个方向，也可以简写（按顺时针方向，上下一样，左右一样）

居中对齐：分为块级元素、行级元素

其他

元素所占实际空间

  • 实际所占宽度=width+左右padding+左右border+左右margin
  • 实际所占高度=height+上下padding+上下border+上下margin

盒子属性默认值

不同标签的盒子属性默认值可能不同，需要自己设置

外边距合并

也称为外边距折叠，指的是两个块级元素垂直外边距相遇时，他们将合并为一个外边距，合并后的外边距值为其中的较大的那个外边距值

两种情况：

  • 当一个元素出现在另一个元素上面时，第一个元素的下外边距与第二元素的上外边距会发生合并

  • 当一个元素包含在另一个元素中时，并且没有内边距或边框把外边距分隔开时，两个元素的上外边距会 发生合并

    外边距合并的好处：让排版在视觉上显得更美观

定位方式

简介

通过position属性实现对元素的定位，有四种定位方式

取值 含义 说明
static 默认值 按照常规文档流进行显示，从上到下，从左到右
relative 相对定位 相对于标签原来的定位进行的定位
absolute 绝对定位 相对于第一个非static定位的父标签的的定位
fixed 固定定位 相对于浏览器窗口进行定位

设置定位方式后，还要设置定位属性（偏移量）：top，bottom，left，right

相对定位

先设置元素的position属性为relative，然后在设置偏移量

绝对定位

先设置父标签为非static定位，然后设置元素的position属性为absolute，最后设置便宜量

注意事项：

  • 一般来说都把父标签设置为非static定位
  • 如果父标签不是非static定位，则会相对于浏览器窗口进行定位
  • 一旦设置元素为绝对定位后，元素会浮在页面上方

固定定位

先设置元素的position属性为fixed，然后设置偏移量

一旦设置元素为绝对定位后，元素会浮在页面上方

z-index

设置元素定位方式后，元素会浮在页面上方，此时通过z-index属性设置优先级，控制元素的堆叠顺序

取值为数字，值越大优先级越高，默认为auto（大多数浏览器默认为0）

注：只能给非static定位的元素设置z-index属性

其他CSS属性

浮动和清除

浮动属性

通过float属性来实现元素的浮动，可以让块级元素脱离常规文档流，向左或者向右移动，在同一行显示，如果一行显示不下，则会换行

常用取值：

  • left：左浮动
  • right：右浮动
  • none：不浮动，默认值

设置float属性后，元素会浮在页面上层，此时父容器无法计算自己的尺寸，通常会在容器末尾添加一个清除了float的空div来解决

清除属性

通过clear属性实现清除，设置元素的哪一侧不允许有浮动的元素，目的为了和其他浮动元素换行隔开，只对块级元素有效

常用属性：

  • left：左清除，左侧不允许有浮动元素
  • right：右清除，右侧不允许有浮动元素
  • both：两侧清除，两侧不允许有浮动元素
  • none：不清除，允许有浮动元素

结论：

  • 对于非浮动元素，两边都可以设置清除
  • 对于浮动元素，向哪边浮动，就只能设置哪边清除

元素的显示和隐藏

display

通过display属性设置元素是否显示，以及是否独占一行

常用取值：

取值 含义 说明
none 不显示
inline 显示为内联元素，行级元素的默认值 块级元素变为行级元素，不再独占一行
block 显示为块级元素，块级元素的默认值 行级元素变为块级元素，独占一行
inline-block 显示为内联元素，但可以设置宽和高 在inline基础上允许设置宽和高
注：行级元素默认无法设置宽和高，可以设置行级元素设置display：inline-block，然后就可以设置宽和高

visibility

也可以通过visibility属性设置元素的显示和隐藏

常用属性：

取值 含义 说明
visible 显示
hidden 隐藏

区别

  • display隐藏时不再占据页面中的空间，后面的元素会占用其空间
  • visibility隐藏时会占据页面中的空间，位置还保留在页面中，只是不显示

轮廓属性

简介

outline：轮廓，用于在元素周围绘制一个轮廓，位于border外围，可以突出显示元素

基本属性

常用属性：

  • outline-width：轮廓的宽度
  • outline-color：轮廓颜色
  • outline-style：轮廓样式
  • outline：简写顺序，与border简写相同

在浏览器中，当鼠标点击或使用Tab键让一个表单或链接获取焦点时，该元素会有一个轮廓outline

  • 优点：可以提高使用表单的用户体验
  • 缺点：有时影响美观

outline与border的区别

  • border可应用与所有HTML元素，而outline主要用于表单元素，超链接等
  • 当元素获取焦点时，会自动出现outline轮廓效果，当失去焦点时会自动消失，这是浏览器默认行为
  • outline不影响元素的尺寸和位置，而border会影响

其他属性

宽高相关属性

  • max-width：设置元素最大宽度
  • min-width：设置元素最小宽度
  • max-height：设置元素最大高度
  • min-height：设置元素最小高度

overflow属性

当元素中内容溢出时该如何处理

常用取值：

  • visible：溢出时可见，显示在元素外面，默认值
  • hidden：溢出部分不可见（常用）
  • scroll：无论是否出现溢出时钟出现滚动条
  • auto：溢出时会自动出现滚动条

cursor属性

用来设置光标形状

常用取值：

  • default：默认光标，一般为箭头
  • pointer：手型，当光标移动到超链接时，一般显示这个
  • move：表示可移动
  • text：表示文本
  • wait：表示程序正忙，需要等待
  • help：表示帮助

表格布局

简介

不使用与复杂的布局，仅适用于简单的，有规则的结构

定位相对准确，与浏览器基本无关，仅适用于简单分割

用法

table常用样式属性

  • border：在表格外围设置边框
  • border-spacing：设置单元格之间的距离（相当于table标签的cellspacing属性，即间距）
  • border-collapse：表格中相邻边框是否合并，取值：seprate，collapse

th/td常用属性

  • border：为单元格设置边框
  • padding：设置单元格的内边距（相当于table标签的cellpadding属性，即边距）

div布局

定位绝对精准，使用灵活，适合于复杂的布局方式

圣杯布局

页面结构：两边的边栏宽度固定，中间主体在一定的范围内可自适应，并且主体优先加载

一般防止页面缩放太小影响浏览，都会为页面设置一个最小宽度min­width

主体部分放在上面先加载

双飞翼布局

源自淘宝的UED（用户体验设计）团队

双飞翼布局和圣杯布局要实现的效果是 相同的，只是思路不同

圣杯布局和双飞翼布局的区别：

  • 双飞翼布局比圣杯布局多创建了一个div
  • 双飞翼布局不用设置边框的相对定位，也不用设置偏移量
  • 双飞翼布局使用的margin，圣杯布局使用的padding
实际开发中建议使用CSS3中的flex弹性盒子布局，更简单
