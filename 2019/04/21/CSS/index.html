CSS简介

什么是CSS

CSS，层叠样式表，是一组样式设置规则，用于空中页面的外观样式

为什么使用CSS

  • 实现内容和样式分离，便于团队开发
  • 样式复用，便于网站的后期维护
  • 可以实现页面的精确控制，可以让页面变得更精美

CSS作用

  • 页面外观美化
  • 布局和定位

CSS基本用法

CSS语法

  • 选择器：要修饰的对象（东西）
  • 属性：修饰对象的哪个属性（样式）
  • 属性值：样式的取值

CSS应用方式

也称为CSS引用方式，有三种方式：内部样式，行内样式，外部样式

内部样式

也称为内嵌样式，在页面头部通过style标签来定义

对当前页面所有符合样式选择器的标签能起作用

行内样式

也称为嵌入样式，使用HTML标签的style属性定义

只对设置style属性的标签起作用

外部样式

使用单独.css文件来定义样式，然后在页面中使用link标签，或者@import指令引入

  • 使用link标签链接外部样式文件（推荐）

    type属性可以省略

    link本质上是链接

  • @import指令导入外部样式表文件

    impot本质上是导入，效率低

选择器

基础选择器

标签选择器

也称为元素选择器，使用HTML标签作为选择器名称

也标签名作为样式应用依据

类选择器

使用自定义名称，以.号为前缀，然后通过HTML标签属性调用类选择器

以标签的class属性作为样式应用的依据

注意事项：

  • 调用时不能添加.号
  • 同时调用多个类选择器中间以空格隔开
  • 类选择名不能以数字开头

ID选择器

使用自定义名称，以#号作为前缀，然后通过HTML标签的id助兴进行名称匹配

以标签的id属性作为样式应用依据，一对一关系

复杂选择器

复合选择器

比如：标签选择器和类选择器，标签选择器和ID选择器，一起使用

必须同时满足两个条件才能应用样式

组合选择器

也称为集体声明

将多个具有相同样式的选择器进行声明，使用,隔开

嵌套选择器

在某个选择器内在设置选择器，通过空格隔开

只用满足层次关系最里层的选择器所对应的的标签才会应用该样式

注：使用空格是不区分父子还是后代，使用CSS中新增的>时必须是父子关系才可以

伪类选择器

根据不同状态来显示不同样式，一般多用于a标签

四种状态：

  • link：未被访问过的链接
  • visited：已经被访问过的链接
  • hover：鼠标移动到的连接上，即悬浮在哪
  • :active：当前正在选定的链接，即激活的链接

注：默认超链接为蓝色有下划线

伪元素选择器

  • first-letter：为第一个字符添加样式

  • firs-line：为第一行添加样式

  • :before：在元素内容最前面去添加其他内容，需要配合content属性使用

  • :after：在元素内容最后面去添加其他内容，需要配合content属性使用

选择器优先级

优先级

ID选择器>类选择器>标签选择器

原因：首先记载标签选择器，再加载类选择器，然后加载ID选择器，最后加载行内样式

内外部样式加载顺序

就近原则

原因：按照书写顺序一次生效，在同优先级的前提下，后加载的会覆盖先加载的同名样式，所以离得越近越优先

!important

可以使用!important使某个样式具有最高的优先级

常用CSS属性

字体属性

定义字体相关样式

属性 含义 说明
font-size 大小，尺寸 可以使用多种单位
font-weight 粗细
font-family 字体
font-style 字体样式
简写

font-size

取值：

  • inherit：继承，默认从父标签继承字体大小（默认值）

    所有CSS属性的默认值都是inherit

  • px：像素

  • %：相对于父标签字体的百分比

  • em：倍数，相对于父标签字体的百分比

    html根元素默认字体大小为：16px，也称为基础字体大小，一般会修改这个默认大小，修改为62.5%（其实就是10px）

font-weight

取值：

  • normal：普通（默认值）
  • bold：粗体
  • 自定义：400为normal，700位bold

font-family

要求系统中安装指定字体

一般建议写3中字体，首选，其次，备用，多个之间以逗号隔开

Arial这个字体肯定有，一般为备用

font-style

取值：

  • normal：普通
  • italic：斜体

font

简写属性：font: font-style | font-weight | font-size | font-family

必须按照这个顺序写

文本属性

属性 含义 说明
color 颜色
ling-hight 行高 行之间的高度
text-align 水平对齐方式 取值：left，center，right
vertical-align 垂直对齐方式 取值：top，middle，bottom，可用于图片与文字的对齐方式
text-indent 首行缩进
text-decoration 文本修饰 取值：underline，overline，line-through，none
text-transform 设置字母的大小写 取值：lowercase，uppercase，capitalize
letter-spacing 字符间距
word-spacing 单词间距 只对英文有效
white-space 空白处理方式 文本超出后是否换行，取值：nowrap

color

取值，四种写法：

  • 颜色名称，使用英文单词

  • 16进制RGB值，#RRGGBB

    特别情况下可以缩写

    不区分大小写

  • rgb函数：rgb(red,green,blue)

    每个颜色的取值范围：[0,255]

  • rgba函数：(red,green,blue,alpha)

    可以设置透明度，alpha取值范围：[0,1]，0表示完全透明，1表示完全不透明

CSS一般写的时候写RGB值，在浏览器开发工具里可以快速找到

背景属性

属性 含义 说明
background-color 背景颜色
background-image 背景图片
background-repeat 用来指定背景图片的重复方式
background-position 背景图片的显示的位置
background-attachment 背景图片是否会跟随滚动 相对于浏览器
background 简写方式

background-color

取值：

  • transparent：透明

background-image

  • 必须使用url()来指定图片地址
  • 如果是在.css样式文件中使用相对路径，此时是相对于css文件，不是相对于HTML文件

background-repeat

取值：

  • repeat：重复（默认）
  • repeat-x：在水平方向重复
  • repeat-y：在垂直方向重复

background-position

默认背景图片显示在左上角

取值：

  • 关键字：top（上面），bottom（下面），left（左边），right（右边），center（中间）
  • 坐标：左上角为（0,0）向右为x的正方向，向下为y的正方向

CSS雪碧图，也称为CSS的Sprites，也称为CSS精灵，是一种CSS图像合并技术

含义：将网页中许多非常小的图标/图片整合到一张大图中，当访问该页面时只需要下载一次，可以减少访问服务器次数，提高性能

原理：使用background-position进行背景定位，使用坐标精确定位出背景图片的位置

background-attachment

取值：

  • scroll：滚动（默认）
  • fixed：固定不动

background

简写属性没有顺序要求，以空格隔开

列表属性

属性 含义 说明
list-style-type 设置列表前的标记
list-style-image 将图像作为列表签的标记
list-style-position 设置标记的位置 取值：outside（默认），inside
list-style 简写

list-style-type

取值：none，disc，circle，square，decimal

此时不再区分有序列表和无序列表，只要设置列表前的标记就可以了

list-style

简写属性：list-style:list-style-type | list-style-image | list-style-position

数学顺序没有要求，也可以只设置一个

列表一般用的不多，导航菜单用的都是列表，ul

表格属性

border-collapse：使表格中相邻的边框是否合并为单一的边框

取值：separate（默认），collapse

使用LiveReload

可以实现当保存页面文件时实时刷新

步骤：

  1. 在Chrome中安装LiveReload扩展程序

    将livereload_2_1_0_.crx文件拖放到Chrome的扩展程序中

    勾选“允许访问文件地址”

  2. 在Sublime中安装LIveReload插件

    将LiveReload.zip解压后放到Sublime的插件目录Packages中

  3. 配置Sublime的LIveReload插件

    首选项 – Package Setting – LIveReload – Settings Default

  4. 在浏览器中启用LIveReload

    先打开要访问的页面，然后点击浏览器地址右边的黑色圆圈，当中心的小圆圈变成实心时表示已启用

  5. 在Sublime中启用LIveReload

    Cirl+Shift+P – LIveReload：Enable/disabled plug-ins – Enable - Simple Reload

盒子模型

简介

盒子模型是网页布局的基础，将页面所有的元素都看作成一个盒子，盒子包含以下几个属性：

  • height：高度
  • width：宽度
  • border：边框
  • padding：内边距，内容距离边框之间的距离
  • margin：外边距，盒子和盒子之间的距离

盒子属性

border

表示盒子的边框

  • 分为四个方向：

​ top（上），right（右），bottom（下），left（左）

​ border-top，border-right，border-bottom，border-left

  • 每个边都包含三种样式：

​ color（颜色），width（粗细），style（样式）

样式style的取值：

​ solid（实线），dashed（虚线），dotted（点线），double（双线），inset（内嵌3D线），outset（外嵌3D线）

简写：三种方式

  • 按方向简写：

    border-top，border-right，border-bottom，border-left

    书写顺序：border-方向：width style color

  • 按样式简写：

    border-color，border-width，border-style

    书写顺序：border-样式：top right bottom left

    必须按顺时针方向书写，同时，是可以缩写的

    ​ border-width:2px; —-> 四个边的宽度都是2px

    ​ border-width:2px,4px —> 上下一样为2px，左右一样为4px

    ​ border-width:2px,3px,4px —> 上面为2px，右面为3px，下面为4px，左面为3px

    规则：如果省略，则认为上下一样，左右一样

  • 终极简写

    如果四边完全相同，border：width style color

padding

表示盒子内边距，及内容与边框之间的距离

同样也分为四个方向，也可以简写（按顺时针方向，上下一样，左右一样）

注：如果上下冲突，以上为准，如果左右冲突，以左为准

margin

表示盒子的外边距，即盒子与盒子之间的距离

同样也分为四个方向，也可以简写（按顺时针方向，上下一样，左右一样）

居中对齐：分为块级元素、行级元素

其他

元素所占实际空间

  • 实际所占宽度=width+左右padding+左右border+左右margin
  • 实际所占高度=height+上下padding+上下border+上下margin

盒子属性默认值

不同标签的盒子属性默认值可能不同，需要自己设置

外边距合并

也称为外边距折叠，指的是两个块级元素垂直外边距相遇时，他们将合并为一个外边距，合并后的外边距值为其中的较大的那个外边距值

两种情况：

  • 当一个元素出现在另一个元素上面时，第一个元素的下外边距与第二元素的上外边距会发生合并

  • 当一个元素包含在另一个元素中时，并且没有内边距或边框把外边距分隔开时，两个元素的上外边距会 发生合并

    外边距合并的好处：让排版在视觉上显得更美观
